# Student Information System (SIS)

This project is developed as part of the **System Programming** course.

## Project Description
The Student Information System (SIS) is a containerized C++ application that demonstrates
basic system programming concepts together with Docker and GitHub workflow.

## Technologies Used
- C++
- CMake
- Docker
- Docker Compose
- PostgreSQL
- Git & GitHub

## System Architecture
- **Application Container**: C++ application built using CMake
- **Database Container**: PostgreSQL database
- **Docker Compose** is used to orchestrate multi-container setup

## How to Run
```bash
docker compose up --build
